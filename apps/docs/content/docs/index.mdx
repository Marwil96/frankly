# Frankly

Frankly is an embeddable product review platform built for headless e-commerce brands. Collect verified reviews via automated post-purchase emails, display them with a clean widget or React SDK, and boost your SEO with structured data.

## Key Features

- **Centra Integration** — Webhook-based, works with Centra's shipment events
- **Post-Purchase Emails** — Automated review requests with configurable timing
- **Photo Reviews** — Customers can upload images with their reviews
- **Multi-Language** — Widget and emails in English, Swedish, Norwegian, Danish
- **SEO Ready** — JSON-LD structured data endpoint for Google rich snippets
- **GDPR Compliant** — Verified purchases, right to erasure, unsubscribe
- **React SDK + JS Widget** — Embed anywhere with clean, brand-neutral styling
- **Public REST API** — Build custom integrations

## How It Works

1. **Connect your store** — Sign up, grab your API key, and configure the Centra webhook
2. **Customers purchase** — When an order ships, Centra fires a webhook to Frankly
3. **Review request sent** — After a configurable delay (default: 7 days), an email invites the customer to leave a review
4. **Review collected** — The customer clicks through, rates the product, and optionally uploads photos
5. **Reviews displayed** — Embed reviews on your product pages via the React SDK, vanilla widget, or REST API
6. **SEO boosted** — The JSON-LD endpoint provides structured data for Google rich snippets
