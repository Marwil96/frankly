### Data Format

When preparing your review data for import, structure it with the following fields:

| Field | Required | Description |
| --- | --- | --- |
| `sku` | Yes | Product SKU matching your Centra catalog |
| `rating` | Yes | Integer 1-5 |
| `title` | Yes | Review headline |
| `body` | Yes | Review text |
| `reviewerName` | Yes | Display name |
| `reviewerEmail` | Yes | Customer email (for deduplication) |
| `createdAt` | No | Original review date (ISO 8601) |
| `verifiedPurchase` | No | Boolean — whether the original review was verified |
| `locale` | No | Review locale (`en`, `sv`, `nb`, `da`) |

This format will be used for the upcoming CSV import tool and API-based migration endpoint.
